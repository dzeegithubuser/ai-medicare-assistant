Analyze a patient's drug list, income, health profile, and location to recommend the best Medicare plans available in their county.

Generation rules:
- Generate exactly 5 ranked Medicare plan recommendations.
- Mix plan types realistically:
  - Include at least 1 MA-PD (Medicare Advantage with drug coverage).
  - Include at least 1 standalone PDP (Part D plan, pair with Medigap if appropriate).
  - Include 1 D-SNP if the patient has dual eligibility indicators (low income + disability).
- Rank by estimated annual total cost (lowest first).

Plan category rules:
- Every plan MUST have a planCategory field. Assign one of these four categories:
  - "MA_ONLY" — Medicare Advantage plan that bundles Part A, B, and D (prescription drug coverage included). This is the all-in-one option.
  - "PDP_ONLY" — Standalone Part D prescription drug plan only. No Medigap supplement paired.
  - "PDP_MEDIGAP" — Part D prescription drug plan paired with a Medigap supplement plan (e.g., Plan G, Plan N). Mention the Medigap plan name in aiExplanation.
  - "MA_PDP" — Medicare Advantage plan that does NOT include Part D, requiring a separate standalone PDP for drug coverage. Mention this in aiExplanation.
- Category assignment must be consistent with planType:
  - planType "MA-PD" → planCategory should be "MA_ONLY" (includes drug coverage) or "MA_PDP" (if the MA plan lacks Part D).
  - planType "PDP" → planCategory should be "PDP_ONLY" or "PDP_MEDIGAP".
  - planType "D-SNP" → planCategory should be "MA_ONLY" (D-SNPs bundle everything).

Plan data rules:
- Use real insurance company names (UnitedHealthcare, Humana, Aetna, Cigna, WellCare, Blue Cross, Kaiser, etc.).
- Plan IDs in format H####-### for MA-PD, S####-### for PDP.
- Star ratings between 2.5 and 5.0 (realistic distribution).
- Premiums: MA-PD $0-$75/month, PDP $7-$85/month.
- Deductibles: MA-PD typically $0, PDP $0-$590.
- MOOP: MA-PD $3,000-$8,300.
- Drug formulary tiers: 1 (preferred generic), 2 (generic), 3 (preferred brand), 4 (non-preferred brand), 5 (specialty).

Drug coverage rules:
- Assign a realistic formulary tier based on the drug type (generic vs brand, specialty vs common).
- Set copay based on tier: Tier 1 $0-$5, Tier 2 $5-$20, Tier 3 $35-$50, Tier 4 $80-$100, Tier 5 25-33% coinsurance.
- Flag prior authorization for specialty/expensive drugs.
- Flag quantity limits where typical (e.g., controlled substances, expensive brands).
- IMPORTANT: Always generate STANDARD copays based on formulary tier. Do NOT reduce copays for LIS eligibility — the system applies LIS adjustments separately after your response.

Pharmacy-aware scoring:
- If selected pharmacies are provided, factor pharmacy network alignment into plan ranking.
- Set hasPreferredPharmacyNetwork to true only when at least one selected pharmacy belongs to the plan's preferred network.
- Plans whose preferred networks include the patient's selected pharmacies should rank higher, all else being equal.
- Mention pharmacy network considerations in the aiExplanation.

Cost calculation:
- estimatedAnnualDrugCost = sum of (monthlyCopay x 12) for all covered drugs.
- estimatedAnnualTotalCost = (monthlyPremium x 12) + annualDeductible + estimatedAnnualDrugCost.
- For PDP plans, note in aiExplanation that a separate Medigap plan may be needed.
- Never recommend a plan where an essential drug is not covered without flagging it prominently.
- planFinderUrl must use the generated planId: https://www.medicare.gov/plan-compare/#/plan-details/{planId}.

Extended plan benefit rules:
- networkType: Assign realistically — MA-PD plans are typically HMO or PPO (some HMO-POS), PDP plans use "N/A", D-SNP are typically HMO.
- includesDental/includesVision/includesHearing: MA-PD and D-SNP usually include these; PDP never does.
- includesFitness: Set true for plans known to offer SilverSneakers or similar programs (~60% of MA plans).
- includesOtc: Many MA and D-SNP plans offer OTC allowances ($20-$120/quarter). PDP plans never do.
- otcAllowancePerQuarter: Realistic quarterly amount ($0 if no OTC benefit).
- gapCoverage: "None" for basic plans, "Some" for plans with partial gap coverage, "Full" for plans covering generics in the gap.
- mailOrderSavings: true if the plan offers lower copays for 90-day mail-order prescriptions.
- providerNetworkSize: "Large" for national carriers (UnitedHealthcare, Humana, Aetna), "Medium" for regional, "Small" for local.
- emergencyCoverage: true for PPO and all plans with nationwide urgent/emergency coverage (most MA plans).
- pros: 2-4 short bullet points highlighting key strengths (e.g., "$0 premium", "Includes dental & vision", "Large provider network").
- cons: 1-3 short bullet points noting trade-offs (e.g., "HMO requires referrals", "High specialty drug copay", "Limited network").
